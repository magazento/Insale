Header Slider

Header slider extension base on FlexSlider jQuery library, it is free open source slider library. This slider supports everything including flash objects and iframes. We also implemented it to admin section.

Ajax Features

Insale template goes with two important ajax features AjaxCart and Ajax Layered Navigation.
They will make your Magento products catalog more friendly and useful without reloading whole page. So your customers will save their time and focus on your products.

Widgets

Template has 3 type of widgets on board. This widgets made as sliders and compatible with all devices and all modern browsers.

  • New Products Slider
  • On Sale Products Slider
  • Reviews Slider

To insert widgets you will need to use built-in magento tool. You can easily insert widget if you click on "Insert Widget" button as shown on image bellow:

Here how widgets looks in raw data

{{widget type="insale/newproducts" template="magazento/product_widget_new.phtml" static_block="new_products" products_amount="10"}}
{{widget type="insale/saleproducts" template="magazento/product_widget_sale.phtml" static_block="sale_products" products_amount="10"}}
{{widget type="insale/reviews" template="magazento/review_widget.phtml" min_rating="5" reviews_amount="10"}}

 

Installation

''Demo Website Way' - Upload demo store package that goes with template to your server, so you will have fully functional store in 2 minutes. 

  1. Extract files from archive and go to "Website Demo" Folder
  2. You will find 2 folder, one is databasand the second is public directore of demo website. You will need upload contents of public directory on your server directory and import database. So in the end you will have files and database installed on your server 
  3. The next step is to change mysql database credentials in local.xml, which is located on your server in /public/app/ directory
  4. Change address setting to your URL in core_config_data "Config Data", web/unsecure/base_url & web/secure/base_url from insale.dev
     
Once installation is done you will be able to acces admin section with following credentials:
 
user: admin123
password: admin123
 
 

'Normal Way' - Install template into your existing magento store. 

  1. Disable your magento cache & clear the store cache under var/cache and all cookies for your store domain
  2. Disable Compilation Mode (if enabled)
  3. Log out from Magento Admin panel if you are logged in
  4. Upload files from /Template Package/public directory to your magento installation
  5. Check that magento has permissions to read tempalate files.
  6. In Admin navigation go to Magazento > Insale Template > Activate Theme. Theme Activation page you will need to set a Store View for themplate, check all checkboxes(for demo block and pages) and click Activate
